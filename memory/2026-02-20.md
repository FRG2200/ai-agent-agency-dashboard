### 🚀 本番デプロイ手順（残タスク）

#### Step 1: Supabase設定（5分）
1. https://supabase.com/dashboard → New Project
2. SQL Editorで `supabase/schema.sql` を実行
3. APIキーを取得

#### Step 2: N8N設定（10分）
1. https://n8n.io/cloud でアカウント作成
2. `n8n/workflow-order-automation.json` をインポート
3. Supabase認証設定
4. Webhook URLを取得

#### Step 3: Railwayデプロイ（5分）
```bash
cd server
railway init --name ai-agent-agency-api
railway variables set SUPABASE_URL=xxx
railway variables set SUPABASE_SERVICE_ROLE_KEY=xxx
railway variables set N8N_WEBHOOK_URL=xxx
railway up
```

#### Step 4: Vercelデプロイ（5分）
```bash
cd client
vercel env add VITE_API_URL  # RailwayのURL
vercel env add VITE_SUPABASE_URL  # SupabaseのURL
vercel env add VITE_SUPABASE_ANON_KEY  # Supabaseのanon key
vercel --prod
```
